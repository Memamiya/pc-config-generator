PC構成表ジェネレーター

HPのカスタムメイドPCの情報をExcelまたはCSVファイルからPC構成表を生成します。

対応形式：Excel (.xlsx, .xls) または CSV (.csv)

Excel/CSVファイルをドラッグ＆ドロップ
または

サンプルデータ形式

以下のような形式でデータを準備してください：

モデル名 プロセッサー OS メモリ ストレージ
HP EliteBook X G1 Intel Core i5 Windows 11 Pro 32GB 512GB SSD

出力される項目

  • モデル名
  • プロセッサー（CPU）
  • OS
  • ディスプレイ
  • Officeソフト有無
  • Webカメラ
  • メモリ
  • ストレージ
  • グラフィックス
  • 光学ドライブ
  • 無線LAN/Bluetooth
  • マウス有無
  • キーボード有無
